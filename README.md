# Калькулятор на TypeScript

Простой браузерный калькулятор, написанный на чистом TypeScript с использованием библиотеки mathjs.

## Возможности

- Базовые математические операции (сложение, вычитание, умножение, деление)
- Процентные вычисления
- Поддержка десятичных чисел
- Управление с клавиатуры
- Современный и красивый интерфейс

## Установка

```powershell
npm install
```

## Запуск

Для разработки с hot-reload:
```powershell
npm run dev
```

Для сборки production версии:
```powershell
npm run build
```

Для сборки в режиме watch:
```powershell
npm run watch
```

## Использование

- Кликайте по кнопкам мышью или используйте клавиатуру
- `Enter` или `=` - вычисление результата
- `Escape` - очистка всего
- `Backspace` - удаление последнего символа

## Технологии

- TypeScript
- mathjs - для безопасных математических вычислений
- Webpack - для сборки проекта
- CSS3 - для стилизации
